# Advocacia Tatiana de Oliveira — Landing Page Profissional

Landing page premium desenvolvida para a **Dra. Tatiana de Oliveira**, advogada especialista em Direito de Família, Civil e do Consumidor, com 17 anos de experiência em Pomerode, SC.

## ⚖️ Sobre o Projeto
Este projeto faz parte do ecossistema **Antigravity**, demonstrando a construção de produtos digitais reais através de orquestração de agentes.

### Características Principais:
- **Design "Lex Aurea"**: Estética refinada com cores Bordô profundo e Dourado Champagne, transmitindo autoridade e distinção jurídica.
- **Performance**: Construída com HTML5, Tailwind CSS 4 e JavaScript Vanilla, garantindo carregamento rápido e fluidez.
- **SEO Local**: Otimizada para buscas em Pomerode/SC e região.
- **Foco em Conversão**: CTAs estratégicos para consulta via WhatsApp e formulário de contato integrado.

## 🛠️ Tecnologias Utilizadas
- **Core**: HTML5, Vanilla JavaScript.
- **Styling**: Tailwind CSS 4 (via CDN).
- **Tipografia**: Cormorant Garamond (Serifa), Raleway (Heading), Lato (Corpo).
- **Interatividade**: Intersection Observer para animações de revelação, sistema de acordeão para FAQ, e contadores animados.

## 📂 Estrutura de Arquivos
- `index.html`: Página única contendo toda a estrutura, estilo e lógica.
- `assets/`: Imagens profissionais, logos e recursos visuais.

## 🚀 Como Executar
Basta abrir o arquivo `index.html` em qualquer navegador moderno ou utilizar um servidor local como o `Live Server`.
